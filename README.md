2. Uma farmácia oferece descontos diferentes em seus medicamentos dependendo da
pessoa que irá usar o medicamento. O desconto difere de acordo com a idade, o sexo
e o histórico de saúde da pessoa: 30% para pessoas com mais de 60 anos; 20% para
crianças com menos de 10 anos; 10% para mulheres e 5% para pessoas portadoras de
doenças crônicas. Os descontos não são cumulativos. Usando padrão de projetos
desenvolva uma aplicação para esse cenário. Explique a escolha do padrão utilizado.
